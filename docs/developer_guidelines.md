# 開發者重點說明 (Developer Notes)

本文檔記錄了 Foxina 專案開發者的核心工作流程與重要資訊。

## 1. Git Worktree 工作流 (Workflow)

本專案使用 Git Worktree 架構來區分開發與生產環境：

- **修改與開發**: 請在 `Foxina-localDev` 目錄下進行。所有的功能新增、樣式調整與邏輯修正應在此完成並測試。
- **發布與提交**: 當確認功能穩定後，切換至 `Foxina` 目錄進行最後的檢查與版本發布。

這樣的架構能確保開發中的不穩定因素不會直接影響到準備發布的分支。

## 2. 管理後台路徑 (Admin Dashboard)

為了優化使用者體驗，我們將詳細的資料庫管理介面從首頁移出。

- **管理後台路徑**: `/dashboard` (即 `www/index.html#/dashboard`)
- **功能**: 包含本地與遠端資料庫的即時搜尋、資料同步狀態查看、以及個別課程的刪除與詳細資訊檢視。
